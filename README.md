# 📌 Overview

AI Interview Simulator is an intelligent interview preparation system that extracts skills from resumes and generates relevant technical interview questions.

The objective is to simulate personalized technical interviews and help candidates prepare for software engineering, AI, and data science roles.

---

# 🚀 Current Features

* Resume Parsing
* Skill Detection
* Personalized Interview Question Generation
* Technical Question Library
* Python Terminal Interface
